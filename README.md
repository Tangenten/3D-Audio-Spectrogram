# 3D Audio Spectrogram in Python, Made with pyqtgraph and pyaudio

Controls
q/e = change xLength
a/s = change yLength
e/r = change xSize
d/f = change ySize
t/y = change height
g/h = change color
u/i = change falloff rate
j/k = change smoothing factor
o/p = change fft scalar
m = toggle auto rotate
